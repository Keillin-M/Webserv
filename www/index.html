Welcome to Webserv

This testing environment allows you to validate all HTTP functionality of your 42 Webserv (C++98) implementation.

Features

  • GET Requests: Test static file serving, directory listing (autoindex), and custom paths
  • POST Requests: Send text data and upload files with full response inspection
  • DELETE Requests: Test file deletion capabilities
  • Redirections: Test HTTP 301, 302, 307, and 308 redirects
  • Error Pages: View custom error pages with http.cat images (400, 403, 404, 405, 413, 500)
  • Response Inspector: See status codes, headers, body, and request duration
