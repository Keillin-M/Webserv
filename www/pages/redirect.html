About HTTP Redirects

HTTP redirects inform the client that a resource has moved to a different location. Different status codes have different semantics:

301 Moved Permanently

Resource permanently moved. Search engines update their links. Method may change to GET.

302 Found (Temporary Redirect)

Resource temporarily at different URI. Original URI should be used for future requests. Method may change to GET.

307 Temporary Redirect

Like 302, but guarantees the HTTP method won't change (POST stays POST).

308 Permanent Redirect

Like 301, but guarantees the HTTP method won't change (POST stays POST).
